### 🚀 Getting Started

**1. Clone the repository**
```bash
git clone https://github.com/karthikbilaspur/beginnner_projects.git
cd beginnner_projects
```

**2. Navigate to any project**
```bash
cd Calculator
python calculator.py
```

**3. Install dependencies if required**
Most projects use only the Python standard library. For projects with external libs:
```bash
pip install -r requirements.txt
```

Check each project's folder for its own `README.md` with specific instructions.
